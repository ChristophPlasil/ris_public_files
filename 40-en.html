Information

If you're unable to view the form for application, your browser may be configured to block all pop-up windows even on the ris.cesni.eu website. While this is a great way to prevent unwanted pop-ups while browsing the web, it can interfere with the pop-up functions.

To overcome this :

  • For internet explorer : Change security and privacy settings for Internet Explorer 11

  • For Chrome : Block or allow pop-ups in Chrome

  • For Firefox : Pop-up blocker settings, exceptions and troubleshooting

  • For Safari (iOS): Block or allow pop-ups in Chrome
